# 🌐 Network Security and Monitoring Analysis

## 📌 Introduction
This project focuses on implementing **Network Security and Monitoring** techniques using open-source tools. It involves analyzing network traffic, identifying suspicious activities, and understanding how systems communicate within a network environment.

---

## 📌 Internship Context

This task was completed as a part of my Vulnerability Assessment and Penetration Testing (VAPT) Internship at Cyart Technologies in March 2026.

---

## 🎯 Objective
- To understand network monitoring fundamentals  
- To analyze real-time and captured network traffic  
- To identify anomalies and potential threats  
- To enhance network visibility and security awareness  

---

## 📍 Scope
- Monitoring network packets  
- Traffic analysis and inspection  
- Identifying unusual or malicious patterns  
- Basic intrusion detection concepts  

---

## 🛠️ Tools Used
- Wireshark (Packet Analyzer)  
- Tcpdump (optional)  
- Kali Linux / Linux Environment  

---

## ⚙️ Methodology

1. **Environment Setup**
   - Installed and configured network monitoring tools  
   - Prepared system for packet capturing  

2. **Packet Capture**
   - Captured live network traffic  
   - Stored packets for analysis  

3. **Traffic Analysis**
   - Inspected protocols (TCP, UDP, HTTP, DNS, etc.)  
   - Filtered relevant packets  
   - Identified communication patterns  

4. **Anomaly Detection**
   - Detected suspicious or unusual traffic behavior  
   - Observed potential threats or irregularities  

5. **Reporting**
   - Documented observations  
   - Highlighted key findings

- Detected unusual packet behavior in specific scenarios
